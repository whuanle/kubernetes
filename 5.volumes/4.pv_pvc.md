# 5.4 PV 和 PVC



## PersistentVolume Claim

对于 PersistentVolume Claim，我们就不写在一起了，我们可以先创建一个存储卷

```yaml
apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: task-pv-claim
spec:
  storageClassName: manual
  accessModes:
    - ReadWriteOnce
  resources:
    requests:
      storage: 3Gi
```

```text
kubectl apply -f pvc.yaml
```

ResourceQuota

```text
hard:
  persistentvolumeclaims:"10"
  requests.storage:"500Mi"
```
